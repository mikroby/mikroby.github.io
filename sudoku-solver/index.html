Sudoku Solver

This app resolves the puzzle you input manually.

Click on any cell to start editing, then scroll

your mouse to select a number or use the num keys.

Click outside the board or choose another cell

to finish editing the current one.

Double-click on a number empties the cell.

By default, an example board is loaded at startup.

'Empty' button clears all the fields.

Click on 'Solve' to start resolving the sudoku board.
